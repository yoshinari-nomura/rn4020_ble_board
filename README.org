* はじめに
  Microchip の RN4020 は，Bluetooth Low Energy 機器作成に必要な機能を1チップに収めた
  非常に使いやすいマイコン製品です．以下の特徴があります．

  + 消費電力非常に小さい
  + 簡易的ながら柔軟なユーザスクリプトを記述できる
  + 特別な開発環境(コンパイラ)の整備が不要
